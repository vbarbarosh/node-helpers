String constants for the type names returned by `gettype`. Covers the
standard `typeof` results (`'null'`, `'undefined'`, `'boolean'`, `'number'`,
`'bigint'`, `'string'`, `'symbol'`, `'function'`, `'object'`) plus the extra
distinctions `gettype` makes: `array` → `'array'`, `nan` → `'nan'`,
`neg_inf` → `'-inf'`, `pos_inf` → `'+inf'`.
